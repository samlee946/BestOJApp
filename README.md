# BestOJ APP 
---

华农刷题宝手机APP  
  
# 开发环境
---

* android studio 1.5.1  
* gradle 2.8  
* Droid4X模拟器
* Eclipse Java EE IDE Helios Service Release 1  
* Tomcat 7.0.69  

# 目录结构
---
doc		存放分类文档  
pic		存放部分软件截图  
project 存放android studio项目  
server 	存放刷题宝服务器的项目  
recommandation	存放推荐算法的相关代码与数据
